Supporting a New Bank

If MT2OFX reports that your file cannot be recognised, you might be able to implement support for your bank yourself. This process consists of several steps, which depend on the format of the file.

 

If your bank provides downloads in MT940 format:

  1. Establish the BIC for your bank.
    If required, you can use the online service to look up your bank.

  2. Define the bank in the Options-->MT940 dialog.

  3. Create a processing script, possibly starting with a copy of an existing script.

  4. Set up a matching rule in the Options-->MT940-->Matching dialog so the origin of the MT940 file is correctly identified and linked to the new processing script.

 

If your downloads are in a CSV or other text format:

  1. Create a processing script, possibly starting with a copy of an existing script.

  2. Add the new processing script to the list of text-processing scripts in the Options-->Text dialog.

  3. By adjusting the sequence of scripts in this dialog, you can ensure that the recognition operates correctly and efficiently by moving the scripts for your banks to the top of the list.
