[AccountTypes] Section

In this section, the value to be used as the Account Type in the output file can be defined for each individual account number.

 

Each entry uses an account type known to OFX; no other types can be used.

 

The OFX account types are shown in the following table.

 

Type

Description  

CHECKING  

Normal Current Account  

CREDITLINE  

Line of credit

SAVINGS  

Savings account

MONEYMRKT  

Money market account

 

The "Default" entry is used for all accounts which are not explicitly typed. It is not usually necessary to change the default or to define individual accounts.

 

Example:

[AccountTypes]

Default=CHECKING

123456789=SAVINGS
