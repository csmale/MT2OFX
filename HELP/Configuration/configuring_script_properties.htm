Configuring Script Properties

This is where you can configure script-specific values (properties) to influence the way that a particular script executes.

 

The dialog looks like this:

 

 

The actual presentation of the value will take on a form which is appropriate for the data type of the selected property. For example, a boolean property will be displayed as a checkbox.

 

Each script can declare which parameters it requires. Some scripts may not use any configurable properties at all.
