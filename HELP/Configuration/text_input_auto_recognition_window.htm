Text Input Scripts Window

The Text Input Scripts window is the place where scripts are configured to handle text (i.e. not MT940) inputs.

 

Each script is programmed to recognise its own files. Before the actual conversion is started, the script to be used must be selected. If the input file name extension is configured in this window, the associated script is selected. Otherwise, the input file is offered to each active script in turn, in the order they appear in this window. The first script which claims to recognise the input is selected to perform the conversion.
