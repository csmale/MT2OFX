MT2OFX.INI

The program configuration and operational parameters are stored in the file MT2OFX.INI which is located in the same directory as the program itself, typically C:\Program Files\MT2OFX.

 

The file is divided into the following sections:

The [General] section

The [AccountTypes] section

The [SpecialPayeeNames] section

The [BankRules] section

The [TxnTypeMap] section

Bank-Specific Configuration sections
