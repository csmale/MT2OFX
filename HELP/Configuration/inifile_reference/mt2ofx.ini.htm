MT2OFX.INI

The program configuration and operational parameters are stored in the file MT2OFX.INI which is located in the user's Application Data directory, typically C:\Documents and Settings\<username>\Application Data\MT2OFX.

 

The file is divided into the following sections:

The [General] section

The [AccountTypes] section

The [QIF] section

The [SpecialPayeeNames] section

The [BankRules] section

The [TxnTypeMap] section

Bank-Specific Configuration sections
