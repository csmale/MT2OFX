[Text] Section

This section contains the list of known text-handling scripts.

 

# list known text input scripts

[Text]

Script1=1,ArgentaNL-TSV.vbs

Script2=0,ING-CSV.vbs

Script3=1,ING-Asc.vbs

 

Note that the script numbers must start from 1 and increase sequentially. The program stops scanning the list if when it encounters a missing number.

 

The first field in the value must be 0 or 1. If the value is 0, the script is inactive and will not be used during the recognition phase. A non-zero value means the script is active and may be called upon.

 

An input file (except MT940 files) is offered to the scripts in order until a script recognises the format of the input file.
