Payee Object

The Payee object contains the information held about a Payee and has the following properties:

 

Name  

Type  

Description  

Name

String

Name of the payee.

Addr1 .. Addr6

String

Space for up to six address lines. Only Addr1-Addr3 are used in OFX; all six can be used in QIF output.

City

String

Not used in QIF output.

State

String

Not used in QIF output.

PostalCode

String

Not used in QIF output.

Country

String

Not used in QIF output.

Phone

String

Not used in QIF output.

LastUsedAddressLine

Integer

Returns the index of the last non-blank address line (1-6). Read-only.
