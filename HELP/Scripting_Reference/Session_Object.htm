Type

Description

FileIn

Full name of the input file.

FileOut

String

Full name of the output file.

BankID

String

Short string identifying the bank.

OutputFileType

String

Possible values: "OFX", "OFC", "QIF". Represents the contents of the file, not simply the extension of the output file name.

PayeeMap

PayeeMap

Payee Mapping information. (from version 3.1)

PayeeMapFile

String

Full name of the payee map data file (normally PayeeMap.Dat) (from version 3.1)

PayeeMapIgnoreCase

Boolean

If True, payee mapping will be case-insensitive. (from version 3.1)

ServerTime

Date

Date/time on the bank's server when the downloaded statement was produced. Often needs to be derived from the file name or its modification time.

Statements

Statements

Provides access to the Statements collection which contains the Statements in the file.
