BankConfig Object

The BankConfig object contains the information loaded from the selected Bank-Specific Configuration and has the following properties:

 

Name  

Type  

Description  

IDString

String

BankName string

Structured86

Boolean

Set to True if the :86: record uses the structured layout

SkipEmptyMemoFields

Boolean

Set to True if empty sections of the transaction description are to be skipped

FindPayeeFn

String

Name of internal function to be called to find the name of the Payee

FindTxnDateFn

String

Name of internal function to be called to find the Transaction Date

FindServerTimeFn

String

Name of internal function to be called to find the time on the server when the file was created

IntuitBankID

String

Intuit Bank ID for use with newer Quicken versions.

ScriptFile

String

Name of the file containing the VBScript functions for processing the data
