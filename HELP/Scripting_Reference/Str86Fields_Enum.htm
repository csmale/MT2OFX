Str86Fields Enum

This enum defines symbols for certain numbered fields in the "Structured :86:" form of MT940 which have particular semantic value.

    sfBatchNum = 10

    sfDetails0 = 20

    sfDetailsLast = 29

    sfBankPayee = 30

    sfAcctPayee = 31

    sfNamePayee = 32

    sfNamePayee2 = 33

    sfTextCodeSupplement = 34

    sfExtra0 = 60

    sfExtraLast = 63

End Enum

 

These values will most frequently be used for retrieving a field using the GetField method of a Str86Info object.
