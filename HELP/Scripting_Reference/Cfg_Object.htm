ProgramConfig Object

The ProgramConfig object contains the program configuration information loaded from the [General] section in MT2OFX.INI and has the following properties:

 

Name

Type

Description

MemoDelimiter  

String   

The character(s) which will be inserted between sections of the transaction description. Normally this will be a semicolon or some similar symbol.

OutputFileType  

String   

Can be any valid file extension, but it is recommended to use only "QIF", "OFX" or "OFC". Any value other than "QIF" or "OFC" causes the output to be in OFX format.

HomePage  

String   

URL of the MT2OFX Home Page.

PayeeCase  

Integer   

0=No Change (default), 1=Upper Case, 2=Lower Case, 3=Proper Case.

AutoBrowse  

Boolean   

If set to True, does a "Browse" (to allow an input file to be selected) as soon as the program is started in GUI mode.

AutoStartImport  

Boolean   

If set to True, the resulting OFX/OFC file is imported automatically into your financial software when the conversion is completed successfully.

PromptForOutput  

Integer   

If set to 0 (default), the program will use the input file name with its extension changed to the value set in "OutputFileType" (see above).

If set to 1, the program will ask you for a file name for the output by means of a "Save As" dialog box.

If set to 2, the program will create a temporary file to hold the output, which is deleted after the conversion.

BookDateMode  

Integer   

0=Book Date (Standard, default), 1=Value Date, 2=Transaction Date (if available, else Book Date)

CompressSpaces  

Boolean

If set to True, multiple consecutive spaces in the name of the payee and the transaction description are replaced by a single space.

ScriptDebugLevel

Integer

Passed through to scripts. e.g. 0=no debugging, 9=max debugging. Other (integer) values can be passed - they are only used in the scripts.

LogFile

String

File to which a simple log is written. Default is no logging. If a relative path (or no path) is specified, the file is relative to the application directory.

TxnDumpFile

String

File to which a dump is written of all statements and transaction data by passing the value to the DumpObjects() function called from the processing script.

ScriptTimeout

Integer

Number of milliseconds to allow for a script function to complete. If the value -1 is specified, no timeout is used. When the value is 0, the timeout occurs as soon as it is determined that a running script is hung for any reason. Default is 5000 (equivalent to 5 seconds).

QifDateFormat

Integer

Value representing the format for dates in QIF output. 0=M/D/Y, 1=D/M/Y, 2=Y/M/D, 3=the setting is taken from the user's regional settings in Windows Control Panel  (default).

QifDecimalSeparator

String

Character to be used as the decimal separator in amounts which are output in QIF files. Can be ".", "," or "" (empty), all without the quotes. If the value is empty or omitted, the setting is taken from the user's regional settings in Windows Control Panel.

QifNoAcctHeader

Boolean

If True, an account header will not be prepended to QIF output. For more information about this option, click here.

ScriptEditor

String

Path to the executable of the program which is launched to edit script files. Default=notepad.exe.

SaveClipboardOutput

Boolean

If True, you will be offered the chance to save the result of the conversion of clipboard data. Otherwise the results are saved to a temporary file which is removed after use. Default is True.

AutoMT940

String

Default: "940,SWI,STA"

AutoText

String

Default: "ASC,CSV,TXT"

String

String
