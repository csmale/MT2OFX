PayeeMapItem Object

(new from version 3.1)

 

Name

Type

Description

Pattern

String

The regular expression which is matched against the payee or memo.

MatchField

Integer

mfPayee (1): match Pattern against the payee

mfMemo (2): match Pattern against the memo

Replacement

String

The string which is used to replace the Payee found when the Pattern matches.

Category

String

The category for the transaction. This information cannot be represented in OFC and OFX files, so is only useful for QIF output.

SIC

Long

Standard Industry Code, defined by the US Government. Used in the USA to classify a company, service or product. It is included in the OFX specification.

NoReturn

Boolean

If True, the rule will not terminate the payee matching process. If False (default) the payee matching stops following the first match.

 

For full information about the regular expression syntax supported in the patterns, see Regular Expression Syntax.
