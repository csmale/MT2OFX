Scripting

In order to ensure that MT2OFX interprets your bank's files as correctly and completely as possible, small pieces of VBScript are used to control the operation.

 

VBScript uses a programming language closely associated with Visual Basic. It has to do without strong typing and many advanced features but remains a powerful mechanism to code "extensions" to a program.

 

Although it is far beyond the scope of this help file to teach you VBScript, you will need to understand MT2OFX's Object Model to be able to access the information within this program. To access Microsoft's VBScript documentation on the MSDN Internet site, click here.

 

The interaction between MT2OFX and the script files is described in How scripts are invoked.
