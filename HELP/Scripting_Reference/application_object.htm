Application Object

The Application object exposes the Visual Basic App object and offers many properties, most of which are uninteresting in the MT2OFX scripting environment. Some of the more useful properties are as follows:

 

Name

Type

Description

Path

String

Path to directory containing the MT2OFX executable

HelpFile

String

Specifies the path and filename of a Help file used by your application to display Help or online documentation.

Title

String

Returns or sets the title of the application that is displayed in the Microsoft Windows Task List

 

For more in-depth information about these and other properties, please refer to the Visual Basic documentation.
